# AdoptaTuÁrbol — Plataforma de Adopción de Árboles

## Introducción
AdoptaTuÁrbol es un sitio web estático desarrollado como proyecto de examen para la certificación Skillnest. Muestra buenas prácticas en HTML semántico, CSS modular y JavaScript funcional, y cumple los requisitos de la rúbrica: filtrado de elementos, interacción de adopción y comportamiento responsive.

## Características
- **Semántica y accesibilidad:** uso de `header`, `nav`, `main`, `footer` y `article` para cada tarjeta; atributos `alt` en imágenes y controles con etiquetas claras.  
- **Interactividad:** filtrado por tipo de árbol y botones de **Adoptar** y **Donar** con respuesta inmediata.  
- **Responsive:** diseño adaptado a escritorio, tablet y móvil mediante media queries y flexbox.  
- **Código organizado:** CSS en `static/css/style.css`, JavaScript en `static/js/script.js` y recursos en `static/images`.  
- **Preparado para validación:** HTML estructurado para pasar la validación W3C.
